### 2023年8月5日
```c
啸霄领域国际基岩版服务器V0.5正式更新:
更新内容:
1.雪球菜单新增转账功能,需要用到经验瓶来操作,另增返回死亡点功能，但此功能还在制作中，暂时不能使用。
2.废除经验瓶，由名为经验能源的光明方块代替，手持抬头即可获得经验，经验能源在经验礼包中获得。
3.新增两个生存区。
4.新增温馨提示，每隔一段时间，玩家下方将会显示温馨提示，告诉玩家关于服务器的一些知识。

6.金苹果现在可以通过附魔台附魔成附魔金苹果，玩家手持一个以上的金苹果，站在附魔台上抬头，通过消耗10级经验得到附魔金苹果。
7.新增隐形精英怪，玩家可以通过发光墨囊让他们显形，手持抬头即可。
8.新增更多矿物addon。
9.主城新增临时挂机点
10.新增感冒系统，玩家在生存的时候有小概率会感染感冒，可以通过牛奶解决。
11.新增货币啸霄币，以后会有新的作用。
12.商店新增部分可购买的物品。
13.新区域建设过半。
14.新主城建设计划启动。
15.有关混战时期玩家死亡赔偿内容更改
(详见赔偿更改公告)
16.修了op功能无法使用的问题
17.Op功能新增op隐身
18.云顶山景区新增保护功能。
19.部分商店物价调整。
20.和平时期与混战时期时间调整。
```

### 2023年7月9日
```c
啸霄领域国际基岩版服务器 v0.4更新内容:
1.商店系统部分完善。(玩家现在可以购买和出售物品，但是只有金币一种货币)
2.新增血月系统:精英怪刷新率增大，所有怪物都有力量buff，苦力怕拥有速度buff。(血月概率为30%)
3.新增公会制度:玩家可以自主创建或加入公会，公会是一整个集体领地，分会长和公会成员，会长拥有使别人加入公会和管理公会、扩张领地、发起战争的权利，公会成员会受到公会保护。想要创建公会的玩家可以向服主或管理员申请(主要是懒得做自动创建公会的功能)。
4.混战时期部分调整:如果在混战时期，受到影响太大从而使心态崩溃，没有继续玩下去的心态时，可以根据实际情况申请向服主或管理索要赔偿。
```

### 2023年7月3日
```c
啸霄领域国际基岩版服务器 v0.3更新内容:
(之前的更新内容都是直接发群里的，现在会以公告的形式发出，以后都会这样)
1.修复雪球菜单不能回城问题。
2.新增在线时间系统(以分钟为单位)
3.新增商店场地(还在完善中)
4.新增和平时期与混战时期倒计时
和平时期:时间为3600秒，即一小时，在此期间，玩家pvp模式会被关闭，死亡不掉落会一直开启，玩家领地保护会一直生效。
混战时期:时间为1800秒，即半小时，在此期间，玩家pvp模式会一直开启，死亡不掉落关闭，玩家领地保护失效，精英怪生成概率增大。
5.啸霄领域信息栏中在线奖励内容由资源奖励改为金币奖励。
以上就是所有更新内容...
全靠服主一个人用指令肝，有谁能帮忙找个指令师的？实在肝不动了😫
```
