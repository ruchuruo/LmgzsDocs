
# 啸霄领域_基岩-1.20.1-基岩服-帮助
---
## 说明
```c
啸霄领域是霄杨和毛衣一起在落梦域服务器创建的领域，
于2020年7月份左右成立，在落梦域的二服和三服都存在的有基地。
现在啸霄领域已成为一个成立国际基岩版服务器，由霄杨担任服主。

霄杨:啸霄领域的创始人之一，现任本群群主、落梦服啸霄领域域长、啸霄领域国际服服主。
毛衣:啸霄领域的创始人之一，现任本群管理、落梦服啸霄领域副域长。
我梦:现任本群管理、落梦域服务器服主。
```

## 问答
```c
Q:啸霄领域国际服与网易落梦服啸霄领域有啥关系及区别？
A:两者之间曾迁移过服务器内的部分建筑，其它与之没有太大的关系。
    网易落梦服啸霄领域是该服务器的其中一个由玩家建立的领域，
    而啸霄领域国际服是一个全新独立的非网易基岩版服务器。

Q:开了啸霄领域国际服，还会管落梦服的啸霄领域吗？
A:会，但大多数都由啸霄领域任定的管理去管，域长现在在忙国际服的事。

Q:为什么要开啸霄领域国际服？
A:主要是因为网易的性能和优化不到位，而且网易租赁服无法添加自定义模组，
    到期就直接销毁存档，导入存档，导出存档等都非常麻烦，
    其次是因为落梦服啸霄领域太冷清了，没有热度。
```
