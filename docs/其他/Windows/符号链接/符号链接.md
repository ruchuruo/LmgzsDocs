# 符号链接

## mklink
### 语法
```cmd
mklink [[/d] | [/h] | [/j]] <link> <target>
```

### 参数
|参数|说明|
|-|-|
|/d        |创建目录符号链接。 默认情况下，此命令创建文件符号链接。|
|/h        |创建硬链接而不是符号链接。|
|/j        |创建目录交叉点。|
|<link\>   |指定要创建的符号链接的名称。|
|<target\> |指定新符号链接所指的路径（相对或绝对）。|
|/?        |在命令提示符下显示帮助。|

### 示例

#### 指向目录的软链接
```cmd
cmd /c mklink /d E:\web\files E:\FILES\files
```
```c
说明
为 E:\web\files <<===>> E:\FILES\files 创建的符号链接
           ↑
    符号链接文件在此文件夹
```
