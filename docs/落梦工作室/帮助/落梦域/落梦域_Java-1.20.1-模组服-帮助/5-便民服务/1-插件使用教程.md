# 插件使用教程

## 玩家互传
::: info TpManager
指令
:::
|缩写指令格式|介绍|
|----|----|
!!tp help            |显示帮助信息, 同!!tp
!!tpa <玩家名字>      |你请求传送到玩家
!!tph <玩家名字>      |请求玩家传送到你
!!tp accept          |同意传送请求
!!tp reject          |拒绝传送请求
!!tp cancel          |取消传送请求
