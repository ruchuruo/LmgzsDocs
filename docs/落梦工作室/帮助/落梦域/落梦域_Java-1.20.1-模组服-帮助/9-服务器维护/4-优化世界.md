# 优化世界
::: tip 适用于 forge
:::
::: info 说明
当 `更新/加/减` 模组后，模组内容会重新分配新的id，
客户端模组内容id和服务端 `world/level.dat` fml 中的id不同时，
会出现模组内容混乱
:::
```c
└─📁world
      📄level.dat
       ├─■ Data
       └─■ fml
         ├─■ Registries
         └─■ LoadingModList
```
::: info 临时解决
- 客户端安装所有模组，包括仅服务端安装的模组，将服务器上的world放到客户端中使用`优化世界`
- 如果点击优化世界出现回到游戏主页，说明模组不全，无法优化
:::
