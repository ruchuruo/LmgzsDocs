::: tip 提示
如果你的文件夹的内容和我的一样，将会是这样的

没有 .minecraft 文件夹需要自行创建
```c
└─📁Minecraft Java
   ├─📁.minecraft
   ├─📁HMCL
   |      📄hmcl.json                              启动器配置文件
   |      ▶️HMCL-<version>.exe                    启动器
   |
   └─📁PCL
       |  ▶️Plain Craft Launcher <version>.exe    启动器
       |
       └─📁PCL                                     启动器配置文件夹
```
![图片](./images/添加游戏目录/添加游戏目录__HMCL_5.png)
