### 你现在的目录应该是这样的
```c
└─Minecraft Java
    ├─.minecraft
    │  │  launcher_profiles.json
    │  │
    │  ├─assets
    │  ├─libraries
    │  └─versions
    │      └─1.20.1_Forge
    │              1.20.1_Forge.jar
    │              1.20.1_Forge.json
    │
    └─HMCL
            hmcl.json
            HMCL-3.5.8.exe
```
