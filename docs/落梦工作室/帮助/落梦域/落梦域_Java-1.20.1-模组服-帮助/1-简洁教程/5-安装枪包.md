## 安装枪包
<br>
将下载的文件放入
<br>
.minecraft\versions\<版本名字>\tacz\



::: tip 注意事项/说明
- 如果只有服务器安装枪包，客户端未安装枪包，则客户端显示 黑紫方块贴图
- 如果只有客户端安装枪包，服务器未安装枪包，则客户端无法使用枪包的物品
- 枪包无法通过服务器自动下载安装，所以只能用户自行下载安装
- 暂时不需要 Packet Fixer 类似的模组
:::
