	性能与安全性优化
	因为新版加密系统使用了C++，所以无法在ARM架构的设备上运行，比如大部分手机用户。
	未安装YSM 的客户端可以加入有 YSM的服务器，反之亦然。

	本次更新主要内容由 @布丁番茄茄 提供，大幅度提升了模组的安全性，极大地强化了模组的渲染性能。
	但由于使用了部分 C++ 代码，导致模组本体文件较大（20 多 MB 是正常大小），且可能会有部分软件误报为病毒。
	该版本将无法在任何 ARM 平台上运行，手机玩家将无法使用！
